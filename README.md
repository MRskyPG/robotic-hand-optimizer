# Оптимизация роботизированной руки

Разрабатывается роботизированная рука, состоящая из кисти и трех пальцевого захвата.  
Оптимизируются:
- геометрии звеньев
- угол расстановки звеньев

**Цель**: максимизировать число контактов (палец-цилиндр) и минимизировать энергопотребление при захвате и удержании.

---

Задачи, которые решали:
1. Собрали модель в MuJoCo
2. Задать параметры приводов
3. Выполнить оптимизацию со следующими границами:
- Длина обеих фаланг L1=[x1] мм.
- Ширина обеих фаланг Wy=[y1] мм.
- Угол расстановки звеньев alpha=[degree1, degree2] рад.
